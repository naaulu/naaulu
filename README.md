# Naaulu

> **Note:** This project is currently a **Work In Progress (WIP)** preview. Features may change and stability is not guaranteed.

**Rainfall estimation toolbox**

## Install

### Windows

```powershell
.\install.ps1
```

### Linux

```sh
./install.sh
```

## Run

```sh
naaulu estimate
naaulu combine
naaulu verify
naaulu plot
```
